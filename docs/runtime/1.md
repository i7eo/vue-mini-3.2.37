# Runtime

## h 函数

h 函数构建 vnode，但是在 vue 中 vnode 有很多类型：

1. Text：文本节点

2. Comment：注释节点

3. Static：静态 dom 节点

4. Fragment：包含多个根节点的片段

5. Element：dom 节点

6. Component：组件

7. [Teleport](https://cn.vuejs.org/guide/built-ins/teleport.html#teleport)

8. [Suspense](https://cn.vuejs.org/guide/built-ins/suspense.html#suspense)
